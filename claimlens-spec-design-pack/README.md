# ClaimLens — Spec & Design Add‑On Pack
Last updated: 2025-11-01

Use this alongside your main ClaimLens repo. This pack adds:
- Contemporary design system (tokens + component styles)
- UX spec with flows, states, and micro‑interactions
- API spec & JSON Schemas for requests/responses/audits
- Enterprise feature addendum with acceptance criteria
- Kiro next-step prompts to generate UI and tasks automatically

## How to use
1) Drop `app/web/design-tokens.css` and `app/web/components.css` into your web app.
2) Review `docs/UX_SPEC.md` and `docs/API_SPEC.md` — align with your PRD/NFR.
3) Feed `docs/KIRO_NEXT_PROMPTS.md` into Kiro to scaffold UI + tasks.
4) Use `docs/FEATURES_ACCEPTANCE.md` as your DoD checklist in PRs.
