# ClaimLens Go Icons

This directory contains icon assets for the browser extension.

## Required Icons

- `icon16.png` - 16x16px - Toolbar icon (small)
- `icon32.png` - 32x32px - Toolbar icon (medium)
- `icon48.png` - 48x48px - Extension management page
- `icon128.png` - 128x128px - Chrome Web Store

## Design Guidelines

- Use the ClaimLens shield logo
- Primary color: Indigo (#4F46E5)
- Background: Transparent or white
- Style: Modern, clean, professional
- Ensure icons are recognizable at all sizes
